# Codveda Data Analytics Internship - Level 1

This repository contains my work for Level 1 of my Codveda Data Analytics Internship.

## Tasks Completed
- Task 1: Data Cleaning - Processed raw stock price data to handle missing values and correct data types using Python.
- Task 2: Exploratory Data Analysis (EDA) - Performed statistical analysis and created visualizations (Heatmaps, Histograms) to identify market trends.

## Technologies Used
- Python 3
- Pandas
- Matplotlib
- Seaborn

## How to Run
1. Ensure you have the dependencies installed: `pip install pandas matplotlib seaborn`
2. Run the cleaning script: `python data_cleaning.py`
3. Run the EDA script: `python eda.py`
